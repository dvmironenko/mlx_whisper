# Инструкция по отправке изменений на удаленный сервер

## Шаги для отправки изменений:

1. **Проверка текущего состояния:**
```bash
git status
```

2. **Добавление изменений в индекс:**
```bash
git add README.md src/main.py src/static/style.css src/templates/index.html
```

3. **Создание коммита (если еще не создан):**
```bash
git commit -m "Добавить раздельный возврат текста и сегментов при включенных временных метках

- Добавлена функциональность создания двух отдельных файлов при включенных временных метках:
  * Основной файл с транскрибированным текстом
  * Файл с информацией о сегментах в формате [start - end] text
- Обновлены JavaScript функции скачивания в веб-интерфейсе
- Добавлены отдельные кнопки для скачивания каждого типа файла
- Обновлена документация в README.md"
```

4. **Отправка изменений на удаленный сервер:**
```bash
git push origin main
```

## Проверка отправленных изменений:

После выполнения команды `git push origin main` все изменения будут доступны в удаленном репозитории.

## Важные замечания:

- Убедитесь, что у вас есть права доступа к репозиторию
- Убедитесь, что вы подключены к интернету
- Если используется двухфакторная аутентификация, вам может потребоваться ввести токен доступа

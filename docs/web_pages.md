# Веб-страницы MLX-Whisper

## Страница списка заданий (`/`)

Главная страница отображает список всех транскрипционных заданий в карточном виде.

### Структура страницы

```

├─────────────────────────────────────────────────────────────────────┤
│  📋 Список заданий                             [+ Новое задание]   │
│  Просмотр и управление сохранёнными транскрипциями                 │
│                                                                     │
│  [Поиск по ID задания...]  [Все задания ▼]                         │
│                                                                     │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │ [📋] Job ID: a1b2c3d4-e5f6-7890-abcd-ef1234567890          │   │

│  │ 📄 test_audio_converted.wav       42.8 MB  [⬇] [🗑]         │   │
│  │ 📄 test_audio.txt                  12.4 KB  [⬇] [🗑]         │   │

└─────────────────────────────────────────────────────────────────────┘
```

### Элементы интерфейса

| Элемент | Описание |
|---------|----------|
| Header | Логотип с названием и переключатель темы (светлая/тёмная) |
| Section Header | Заголовок "Список заданий" с кнопкой "Новое задание" |
| Filters | Поле поиска по ID и выпадающий список периода (все/7 дней/30 дней) |
| Job Card | Карточка задания с полным Job ID |
| File Row | Файл с иконкой, именем, размером и кнопками действий |
| Job Actions | Кнопка "Удалить задание" (удаляет всю папку) |
| Card Footer | Дата создания и общий размер всех файлов |

### Структура карточки задания

Карточка состоит из 4 частей:

1. **Заголовок** - иконка и полный Job ID (без сокращения)
2. **Файлы** - список файлов задания:
   - Иконка по типу (аудио/текст/видео)
   - Имя файла
   - Размер (в B/KB/MB/GB)
   - Кнопки: Скачать (зелёная), Удалить (красная)
3. **Job Actions** - кнопка "Удалить задание" (удаляет всю папку)
4. **Footer** - дата создания и общий размер

### Функционал

- **Загрузка данных** - API вызов `GET /api/v1/jobs`
- **Поиск** - фильтрация карточек по частичному совпадению ID
- **Период** - отображение только заданий за последние 7 или 30 дней
- **Скачать файл** - `GET /api/v1/files/{filename}/download`
- **Удалить файл** - `DELETE /api/v1/jobs/{job_id}/files/{filename}`
- **Удалить задание** - `DELETE /api/v1/jobs/{job_id}` (удаляет всю папку)

## Страница загрузки (`/uploads`)

Страница для создания новых заданий транскрипции.

### Форма загрузки

- **Выбор файла** - drag&drop или клик для выбора
- **Параметры транскрипции**:
  - Язык (Авто, Русский, English, Deutsch, Français, Español)
  - Задача (Транскрипция, Перевод на английский)
  - Модель (Tiny, Base, Small, Medium, Large, Turbo)
  - Дополнительные настройки (пороги, удаление тишины и т.д.)
- **Кнопки**: Транскрибировать, Список заданий

## API Endpoints

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/v1/jobs` | Список всех заданий |
| GET | `/api/v1/jobs/{job_id}` | Статус задания |
| DELETE | `/api/v1/jobs/{job_id}` | Удалить всё задание (папку) |
| DELETE | `/api/v1/jobs/{job_id}/files/{filename}` | Удалить файл из задания |
| GET | `/api/v1/files/{filename}/download` | Скачать файл |
| POST | `/api/v1/transcribe` | Создать новое задание |

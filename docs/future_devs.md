# Не реализованная функциональность проекта

## Отчет психолога о сессии
Пользователь может создать отчет для каждого задания нажав на кнопку "Создать отчет" расположенную в Job Actions карточки задания.
Использовать сервер OpenAI http://studio2.mironenko.org:8880/v1, API_KEY=1234. Модель "Qwen3.5-9B-MLX-8bit"

Порядок создания отчета:
  - Отчет создать в файле [Отчет по заданию {UUID}.md]
  - Создать "шапку" отчета с даты и UUID задания
  - Вставить заголовок: Расшифровка беседы
  - Создать расшифровку диалога с разбивкой по спикерам в виде [Спикер:] [Высказываение]. Контекст: содержание текстового файла с таймкодами, промт взять из переменной окружения REPORT_TRANSC  
  - Вставить ее в файл отчета.
  - Вставить заголовок: Резюма беседы
  - На основаниии расшифровку диалога с разбивкой по спикерам
